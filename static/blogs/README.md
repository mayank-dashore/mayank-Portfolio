# Blog Content Directory

This folder is for storing blog articles and related assets.

## Structure
```
/blogs
  ├── /blog-1-data-pipeline-architecture
  │   ├── index.md
  │   └── assets/
  ├── /blog-2-power-bi-semantic-models
  │   ├── index.md
  │   └── assets/
  └── /blog-3-aws-data-solutions
      ├── index.md
      └── assets/
```

## How to Add a Blog

1. Create a new folder for each blog post
2. Name it descriptively (e.g., `blog-1-data-pipeline-architecture`)
3. Create an `index.md` file with your blog content (Markdown format)
4. Store any images or assets in an `assets/` subfolder
5. Update the navigation links in `templates/index.html` when ready to publish

## Sample Blog Structure

Blog articles can include:
- Title
- Publication date
- Author information
- Main content with code snippets
- Images and diagrams
- Related links

You can use any static site generator or manual HTML rendering to display these blogs.
